# puzzle-array

Responds to the shuffled array.

## install

### npm

```
npm install puzzle-array
```

### unpkg

```
<script src="https://unpkg.com/puzzle-array" type="text/javascript"></script>
```

## Example

```
let array = [0, 1, 2, 3, 4, 5]
array = puzzleArray(array)
console.log(array) // [3, 0, 5, 4, 1, 2]
```
